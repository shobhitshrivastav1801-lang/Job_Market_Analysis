# 📊 Job Market Analysis using SQL & Power BI

## 📌 Overview
Analyzed **742 job postings** to identify hiring trends, salary patterns, industry demand, required technical skills, and the impact of education on compensation. The project delivers business insights to support recruiters, employers, and aspiring data professionals in making data-driven decisions. :contentReference[oaicite:0]{index=0}

## 🛠️ Tools Used
- MySQL
- Power BI
- Microsoft Excel
- Git & GitHub

## 📂 Project Pipeline
Job Market Dataset (CSV)
→ Data Validation & Quality Checks
→ SQL Analysis
→ Business Insights
→ Power BI Dashboard
→ Recommendations

## 📊 Key Highlights
- Analyzed **742 job postings** from **343 companies** across **60 industries**.
- Solved **9 business analysis questions** using SQL.
- Built an **interactive Power BI dashboard** with KPIs, hiring trends, salary analysis, and skill demand.
- Generated actionable insights on hiring locations, salary distribution, education impact, and in-demand technical skills. :contentReference[oaicite:1]{index=1}
